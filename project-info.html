💰 Project 3 – Coin Calculator Server

This project involved building a backend Express.js server that calculates either:

  • All possible coin combinations for a given cent value
  • The total dollar value for user-input coin counts

I used modular JavaScript (`module.exports`) to keep the logic reusable and maintainable. The frontend HTML lets users explore the coin functionality through a clean form interface.

🛠️ Technologies Used

  • Node.js & Express
  • JavaScript modules
  • Fetch API
  • HTML, CSS (basic frontend)

📁 Files Included

  • p3-server.js – Express backend server
  • p3-module.js – Logic for coin combination and value
  • index.html – Client interface to interact with the server
  • package.json – Declares dependencies
